# Gridstrap v1.0
Framework with CSS grid

Documentation: https://gridstrap.netlify.com

## Features
- Layout with Grid CSS
- No Column Layout
- Cards Scroll-x Mobile
- Menu Navbar Fullscreen

## Usage

### CSS
```html
<head>
  <link rel="stylesheet" href="gridstrap.min.css">
</head>
```

### JavaScript
```html
<script src="gridstrap.js"></script>
```
